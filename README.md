# 🚀 EazyFlare - Telegram Bot & Cloudflare Mini App

EazyFlare is a tool to manage your Cloudflare DNS records via Telegram Mini App.

---

## 🇬🇧 English
### 🛠️ Setup
1. Deploy `worker.js` to Cloudflare Workers.
2. Host `webapp.html` and point its `WORKER_URL` to your worker.
3. Set your `BOT_TOKEN` in `bot.py`.
4. Run `python3 bot.py`.
